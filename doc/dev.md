launch a command and lookup profile:

`firewrap cmd`

`firewrap cmd --arg`

with symlink

`cmd`

`cmd --arg`

pass options, e.g. to override profile, `--` mandatory

`firewrap --profile aprofile -- cmd --arg`

ability to pass different arguments to command?

`cmd --profile aprofile -- --arg`

same as

`firewrap --profile aprofile -- cmd --arg`
